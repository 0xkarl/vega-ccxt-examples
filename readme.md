## examples demonstrating vega integration in [ccxt](https://github.com/ccxt/ccxt) library

### supported methods

- fetchMarkets
- fetchCurrencies
- fetchTickers
- fetchTicker
- fetchOrder
- fetchOrders
- createOrder
- amendOrder
- cancelOrder
- fetchPositions

## getting started

- copy `.env.sample` to `.env` and update accordingly
- install node packages: `yarn`
- run bot: `npm run start`
